Editorial Citrus と Newsreader の組版

2026-05-08 ・ Typography Phase 0 ・ 36rem / line-height 1.85

紙のような暖色クリーム背景に、Newsreader VF の Serif で 本文を組みます。ここに italic や bold を混ぜつつ、欧文 (Newsreader) と和文 (システム明朝 or システム Sans) のベースラインがどれだけ揃うかを観察します。

和欧混植のベースライン

日本語の文章中に English words が 突然差し込まれた場合 でも、漢字・かな・欧文の 高さが揃って読めることが Editorial の前提です。Newsreader の x-height はやや高めで、和文の漢字フェイスと相性が変わります。

数字・約物

09:42 — 2026/05/07 ・ ¥12,800 ・ (No. 359) ・ <Tokyo> ／「日本語」“English” ‘mixed’ 〜 …。

小サイズの読み味

14px に縮めたメタ的な文章でも、にじみ・つぶれが少なく読めるか どうか。Newsreader は opsz 軸を持つので、本来は 14px 以下では 表示用 opsz を切り替えて滲みを抑える設計です (本実装で採用予定)。

コードブロック (JetBrains Mono)

const editorial = {
  font: "Newsreader",
  weight: 400,
  lineHeight: 1.85,
  maxWidth: "36rem",
};

※ コードブロック内に和文を入れない方針 (CLAUDE.md に明記)。 やむを得ず和文を入れる場合は OS の monospace フォールバックで 揃わない見栄えを許容してください。
